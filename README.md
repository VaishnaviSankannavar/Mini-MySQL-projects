# Mini MySQL Projects

A collection of beginner-level mini projects built using MySQL to practice database design, SQL queries, and data management.

## 📌 Overview

This repository contains small database projects created using MySQL. These projects help in understanding core database concepts such as table creation, relationships, joins, and CRUD operations.

The goal of this repository is to improve SQL skills and demonstrate practical usage of relational databases.

## 🛠 Technologies Used

* MySQL
* SQL
* Database Design

## 📖 Concepts Practiced

* Database schema design
* Table creation
* Primary and foreign keys
* Joins
* CRUD operations (Create, Read, Update, Delete)
* Aggregate functions
* Data filtering using WHERE clause

## 🚀 How to Use

1. Install MySQL.
2. Download or clone this repository.
3. Import the SQL files into your MySQL server.
4. Run the queries to create tables and insert data.

## 🎯 Purpose

These projects are created for learning and practicing SQL and database concepts.
